📋 Como Editar o Site — Guia Simples

Este guia foi feito para quem não sabe programar. Sem complicação — só siga os passos!

💡 Dica importante: Sempre faça uma cópia de backup do arquivo antes de editar. Se errar, você terá o original para restaurar.

🖥️ 1. Qual programa usar para editar

Você vai editar o arquivo the-isle-evrima.html com um editor de texto simples. Os melhores são:

⭐
Notepad++ (Recomendado — gratuito) Fácil de usar, mostra o código colorido e tem busca avançada. Baixe em: notepad-plus-plus.org
🆓
Bloco de Notas do Windows (já vem instalado) Funciona, mas é mais difícil. Use apenas se não tiver outra opção. Clique direito no arquivo → Abrir com → Bloco de Notas.
⚠️ Nunca abra com o Word ou LibreOffice — eles vão estragar o arquivo.

📂 2. Como abrir o arquivo para editar

1 Clique com o botão direito no arquivo the-isle-evrima.html
2 Clique em "Editar com Notepad++" (ou "Abrir com" → Notepad++)
3 O código vai aparecer. Não se assuste — você só vai mexer nos textos!
4 Para encontrar algo específico, use Ctrl + F e digite o que quer achar
5 Depois de editar, salve com Ctrl + S
6 Para ver o resultado, dê dois cliques no arquivo HTML e ele abre no navegador

✏️ 3. O que você pode editar facilmente

📝 Mudar textos do site

Use Ctrl + F para buscar o texto que quer mudar e substitua diretamente. Exemplos:

O seu guia definitivoO subtítulo do hero
CostisaqueSeu nome na seção Equipe

Nova Mecânica de Correntes de ArTítulo de uma atualização
DiscordLink do Discord no footer

🦖 Editar os dinossauros

No arquivo, procure por const dinos=[ — ali estão todos os dinos. Cada um tem esta estrutura:

{
  name:'Tyrannosaurus Rex',   ← Nome do dino
  type:'carnivoro',             ← Tipo: carnivoro / herbivoro / aquatico
  icon:'🦖',                    ← Emoji que aparece no card
  cls:'Carnívoro · Apex',       ← Subtítulo do card
  badge:'carn',                 ← carn / herb / omni
  bl:'Carnívoro',               ← Texto do badge
  desc:'O rei incontestável...', ← Descrição
  stats:{
    Vida:95,      ← Número de 0 a 100
    Ataque:98,
    Velocidade:55,
    Furtividade:30
  }
},
💡 Para adicionar um novo dino, copie um bloco inteiro (de { até },), cole depois do último dino e mude os dados.
⚠️ Não apague as vírgulas, as aspas ou as chaves { } — isso quebra o site.

🗺️ Editar as regiões do mapa

Procure por Costa Leste — você vai ver os blocos de cada região. Cada um tem:

  • Um emoji (ex: 🌊)
  • Um nome (map-loc-name)
  • Uma descrição (map-loc-desc)
  • Uma dificuldade (map-loc-tag)

Basta trocar o texto entre as tags. Exemplo: mude Costa Leste por outro nome que quiser.

📰 Editar as atualizações

Procure por update-card — cada card tem:

  • update-date → a data
  • update-title → o título
  • update-excerpt → o texto da notícia
  • update-tag → a etiqueta colorida (ex: Patch Notes)

👥 Editar a equipe

Procure por team-card. Cada membro tem:

  • team-avatar → emoji de avatar
  • team-name → nome
  • team-role → cargo
  • team-bio → bio curta

📚 Editar os guias

Procure por guia-card. Cada guia tem:

  • guia-icon → emoji
  • guia-title → título
  • guia-desc → descrição

🎨 4. Mudar cores (opcional)

No topo do arquivo, procure por :root { — ali estão todas as cores. As principais são:

💡 Para mudar uma cor, basta substituir o código hex (ex: #6db528) por outra cor. Use o site htmlcolorcodes.com para pegar códigos de cor.

🔗 5. Adicionar links reais (Discord, etc.)

Procure por href="#" no arquivo — cada um desses é um link vazio. Para colocar um link real, troque o # pela URL completa. Exemplo:

Antes:  <a href="#">Discord</a>
Depois: <a href="https://discord.gg/seuservidor" target="_blank">Discord</a>

🖼️ 6. Adicionar imagens

Para adicionar uma imagem ao site, coloque o arquivo de imagem na mesma pasta que o HTML e use este código onde quiser:

<img src="nome-da-imagem.jpg" alt="Descrição da imagem" style="width:100%; border-radius:8px;">
⚠️ Use imagens em .jpg ou .png. Imagens muito grandes deixam o site lento — tente usar arquivos abaixo de 500KB.

💾 7. Fluxo de trabalho recomendado

1 Faça uma cópia do arquivo antes de mexer (the-isle-evrima-backup.html)
2 Abra o original no Notepad++
3 Use Ctrl + F para achar o texto que quer mudar
4 Faça as alterações
5 Salve com Ctrl + S
6 Dê dois cliques no HTML para abrir no navegador e conferir
7 Se ficou bom → suba para o GitHub Pages. Se errou → restaure o backup
